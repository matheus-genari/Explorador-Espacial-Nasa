## 📌 Sobre o projeto

O **Explorador Espacial** é uma aplicação web que consome a **API oficial da NASA (APOD – Astronomy Picture of the Day)**, permitindo que o usuário visualize imagens astronômicas de acordo com uma data selecionada.

A aplicação realiza validações, exibe informações completas da mídia retornada (título, descrição e imagem/vídeo) e armazena automaticamente as consultas realizadas utilizando **LocalStorage**, exibindo-as em uma tabela dinâmica.

---

## 🚀 Tecnologias utilizadas

- **HTML5** – Estrutura da aplicação  
- **CSS3** – Estilização e layout responsivo  
- **JavaScript (ES6)** – Lógica e interatividade  
- **NASA APOD API** – Fonte das imagens astronômicas  
- **LocalStorage** – Persistência de dados no navegador  
- **Git & GitHub** – Versionamento e publicação  
- **GitHub Pages** – Hospedagem do projeto  

---

## 🔎 Funcionalidades

- Buscar imagem astronômica por data  
- Validação de datas (não permite datas futuras ou inválidas)  
- Exibição de título, descrição e imagem/vídeo  
- Histórico de pesquisas salvo automaticamente  
- Tabela dinâmica com consultas salvas  
- Exclusão de consultas com confirmação  
- Opção de tradução da interface (PT / EN)  
- Interface responsiva e temática (NASA)
